title: Installation
description: Install Wraith Browser in under a minute
---

## Build from source

```bash
git clone https://github.com/suhteevah/wraith-browser
cd wraith-browser
cargo build --release
```

Requires Rust 1.78+. The binary is ~15MB and lands in `target/release/wraith-browser`.

## Verify

```bash
./target/release/wraith-browser --version
```
