## The 3-command scrape

### 1. Navigate

```json
{
  "tool": "browse_navigate",
  "arguments": { "url": "https://news.ycombinator.com" }
}
```

Response: `{ "url": "https://news.ycombinator.com", "title": "Hacker News" }`

### 2. Snapshot

```json
{
  "tool": "browse_snapshot"
}
```

Returns the DOM with `@ref` IDs on every interactive element — these are how you target clicks and fills.

### 3. Extract

```json
{
  "tool": "extract_markdown"
}
```

Returns the page content as clean Markdown — headings, links, lists, all preserved.

## What just happened

1. Wraith navigated to the URL using its native html5ever-based engine (~50ms)
2. The snapshot mapped every element to an `@ref` ID for agent interaction
3. The extractor converted the rendered DOM to structured Markdown

No Chrome was launched. No browser driver was installed. The entire operation used ~8MB of memory.
